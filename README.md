# 5970_Assignment3
A simple 2D shooter game made in Unity.
____________________________________________
Controls: A/D or left/right arrow to move, click to shoot.
____________________________________________
Objective: Avoid the meteors and bullets and shoot the enemies.
If the game does not appear, make sure you go to the Scenes folder and select Sample Scene
